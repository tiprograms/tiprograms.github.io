Installation Instructions

How to install programs onto your TI-84 calculator

Compatible Calculators

  • TI-84 Plus CE
  • TI-84 Plus C Silver Edition
  • TI-84 Plus
  • TI-83 Premium CE

Some simpler programs may run on TI-83 and older versions

What You'll Need

Before you begin, make sure you have:

  • Your compatible calculator
  • A Mini-B USB cable (usually comes with your calculator)
  • A computer with TI Connect CE software installed (intructions below)
  • The program files (.8xp) downloaded from our site

Step 1: Download TI Connect CE

TI Connect CE is the official software for transferring files to and from your TI calculator. Follow these steps to download and install it:

  1. Click this link to go directly to the TI Connect CE download page: TI Connect CE Download
  2. Click the "Download" button on the page
  3. Download the appropriate file for your computer's operating system (Windows or MacOS)
  4. Run the installer and follow the on-screen instructions
  5. Restart your computer if prompted

Tip: TI Connect CE is a completely free and official software provided by Texas Instruments and works on both Windows and Mac.

Step 2: Connect Your Calculator

Establishing a connection between your calculator and computer is essential for transferring programs. Here's how to do it safely:

  1. Connect your TI-84 Plus CE to your computer using the USB cable
  2. Turn on your calculator
  3. Open TI Connect CE software
  4. Wait for the software to recognize your calculator

Connection Tips:

  • Make sure the USB cable is firmly connected to both devices
  • If your calculator doesn't appear, try a different USB port
  • Some computers may require you to install drivers - TI Connect CE usually handles this automatically
  • The calculator should show "Connected" or a USB icon when properly linked

Step 3: Transfer the Programs

Once your calculator is connected, you can transfer .8xp program files using one of these methods:

Method 1: Using the Program Editor

  1. In TI Connect CE, click on "Program Editor" from the main menu
  2. Click "Open" and navigate to your downloaded .8xp file
  3. The program will open in the editor window
  4. Click "Send to Calculator" or use Ctrl+T (Cmd+T on Mac)
  5. Select your connected calculator from the dropdown menu
  6. Click "Send" and wait for the transfer to complete

Method 2: Drag and Drop

  1. In TI Connect CE, click on "Calculator Explorer"
  2. You'll see your calculator's file system in the right panel
  3. Open a file browser window and navigate to your .8xp files
  4. Simply drag the .8xp files from your computer to the calculator window
  5. Drop them in the calculator's file area
  6. The files will automatically transfer - you'll see a progress bar
  7. Once complete, the programs will appear in your calculator's program list

Transfer Notes:

  • Larger programs may take longer to transfer
  • Don't disconnect the calculator during transfer
  • You can transfer multiple programs at once

Step 4: Running Programs on Your Calculator

After successfully transferring programs to your calculator, here's how to access and run them:

  1. Press the PRGM button on your calculator
  2. Navigate to the program you want to run using the arrow keys
  3. Press ENTER to select it
  4. Press ENTER again to run the program

Program Navigation Tips:

  • Programs are listed alphabetically in the PRGM menu
  • You can also type the first letter of a program name to jump to it quickly
  • Some programs may require specific input - read any on-screen prompts carefully
  • To exit a running program, press ON or follow the program's exit instructions

Troubleshooting

Calculator Not Recognized

If TI Connect CE doesn't detect your calculator, try these solutions:

  • Check that the USB cable is properly connected to both devices
  • Try a different USB port on your computer
  • Restart both TI Connect CE and your calculator
  • Update TI Connect CE to the latest version
  • On Windows, check Device Manager for any driver issues
  • Try a different USB cable if available
  • Ensure your calculator's battery isn't too low

Program Won't Run

If a program doesn't execute properly, consider these fixes:

  • Verify the program is compatible with your calculator model
  • Check that the program transferred completely (re-transfer if needed)
  • Ensure you have the latest calculator OS version
  • Some programs require specific libraries or dependencies
  • Reset your calculator if programs consistently fail to run
  • Check the program documentation for special requirements

Important: Make sure your calculator has enough memory to store the programs. Delete unused programs or data if needed.
